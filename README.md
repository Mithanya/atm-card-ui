# ATM Card UI Design

This project demonstrates a realistic ATM card user interface built using HTML and CSS. The design replicates the visual appearance of a physical debit/credit card through precise layout, typography, gradients, and shadow effects.

The interface includes essential card elements such as the chip, card number, expiry date, cardholder name, and network branding. The focus of this project is on frontend design accuracy and visual detailing without the use of JavaScript.

## Technologies Used
HTML5  
CSS3 (Flexbox, Gradients, Shadows, Media Queries)

## Features
Realistic card layout and structure  
Clean and consistent typography  
Gradient-based styling for depth and texture  
Subtle shadow effects for a physical card appearance  
Responsive design for multiple screen sizes  
Pure HTML and CSS implementation
